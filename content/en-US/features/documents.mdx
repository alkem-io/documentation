# Documents

Documents lets your community create and collaborate on spreadsheets, presentations, and text documents, directly inside a Space. Multiple members can work on the same document at the same time and see each other's changes live.

## What Can You Do with Documents?

- **Co-edit in real time**: work together on a document without emailing files back and forth.
- **Three document types**: create a text document, a spreadsheet, or a presentation.
- **Stay in context**: documents open within the Space, so the content stays connected to your collaboration.
- **See who is editing**: a save indicator at the bottom of the editor shows your current save state and the avatars of other members who have the document open.

## Creating a Document

Any member with contribute permission can create a new document:

1. Open a **Documents** callout in the Space.
2. Click **Create**.
3. Enter a title and select the document type (text document, spreadsheet, or presentation).
4. The document opens in the editor. Start collaborating straight away.

## Opening and Editing a Document

Click any document card to open it. If you have edit rights, the document opens in edit mode. If you have read-only access, it opens in view-only mode.

While editing, a save indicator at the bottom of the editor shows your current state: *Saved*, *Saving…*, *Unsaved changes*, or *Error saving*. Changes are saved automatically. You can also trigger a manual save.

If the editor does not load, it may be temporarily unavailable. Try again in a few minutes.

## Renaming and Deleting Documents

- **Rename**: open the document settings and change the title. The document name stays in sync with the callout title.
- **Delete**: select the delete option and confirm. Deletion is permanent.

## Access and Permissions

All members of a Space can view and open documents. Whether members can also create and edit documents depends on the Space settings. Space admins can control this via **Settings > Allowed Actions > Create Posts**. When this is turned off, only admins can add or edit documents.

## Tips

- Use **text documents** for meeting notes, project descriptions, or any written content the team works on together.
- Use **spreadsheets** for tracking tasks, budgets, timelines, or structured data.
- Use **presentations** for proposals, overviews, or visual summaries to share with the community.
- Add multiple Documents callouts to different tabs of a Space to keep content organised by topic.
