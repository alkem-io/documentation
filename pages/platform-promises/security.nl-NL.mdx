# Veiligheid

Voor meer informatie over de veiligheid op Alkemio, raadpleeg ons verantwoordelijk openbaarheidsbeleid **[op onze website](https://welcome.alkem.io/security/)**.

### Wil je een beveiligingskwetsbaarheid melden?

Neem contact op met **[security@alkem.io](mailto:security@alkem.io)** om het Alkemio Security Team op de hoogte te stellen.

### Heb je extra hulp nodig?

Stuur ons een [email](mailto:info@alkem.io) of bezoek [onze website](https://welcome.alkem.io/contact).
