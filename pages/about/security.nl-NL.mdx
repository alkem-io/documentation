# Security

For more information about security on Alkemio, please refer to our Responsible Disclosure policy **[on our website](https://welcome.alkem.io/security/)**.

### Are you reporting a security vulnerability?

Please contact **[security@alkem.io](mailto:security@alkem.io)** to inform the Alkemio Security Team.

### Do you need additional help?

Send us an [email](mailto:info@alkem.io) or visit us on [our website](https://welcome.alkem.io/contact).
